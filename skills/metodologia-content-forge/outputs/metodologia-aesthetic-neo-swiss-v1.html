Anti-patrones · top-5

Lista roja visual · 0 tolerancia

No hacer

  • Fotos stock realistas
  • Texturas raster (papel arrugado, grunge)
  • Colores fuera de paleta canónica
  • Sombras profundas (drop-shadow ≥10px)
  • Mezcla de estilos icono (line + solid + 3D)
  • Texto sobre fondo ruidoso
  • Verde dominante (#42D36F, #00FF00)
  • Más de 3 sizes tipográficos por pieza

Sí hacer

  • Ilustración vectorial flat
  • Geometría limpia · radius suaves
  • Solo 6 colores canónicos + neutrales
  • Micro-shadow suave (≤6px blur)
  • Un solo estilo de icono por pieza
  • Texto sobre canvas plano + alto contraste
  • Lilac para temas humanos
  • Max 3 sizes tipográficos por pieza

Validación brand visual · checklist

  • Solo 6 colores canónicos + neutrales/derivados oficiales
  • Tipografías: Poppins (head) + Montserrat (body) + Trebuchet (notes)
  • Max 3 sizes tipográficos por pieza
  • WCAG AA mínimo (AAA preferido): 4.5:1 / 7:1
  • Spacing en múltiplos de 8px
  • Cards con shadow suave (≤6px blur)
  • Iconos consistentes (un solo estilo por pieza)
  • 0 imágenes raster realistas en hero
  • 0 colores fuera de paleta
  • 0 verde dominante
  • Footer presente con MetodologIA · year · license
  • Footer NO usa "gratis" ni derivados (consistente con voz v4)
